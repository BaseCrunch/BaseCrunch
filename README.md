## 🎥 Playback Pipeline (THIS IS THE SECRET SAUCE)

```bash
[User Click]
     ↓
[Preload Episode]
     ↓
[Front Probe Validation]
     ↓
[Full Teardown Old Session]
     ↓
[Create New Playback Session]
     ↓
[Start Stream]
     ↓
[Autoplay → Next Episode]
```

💡 **Key Philosophy:**
- ❌ No reused media state  
- ❌ No “fake ready” playback  
- ✅ Every episode = fresh system  

---

## 🧩 Architecture (V2)

```
EpisodePlaybackSession
 ├── Torrent Session
 ├── Preload Controller
 ├── Playback Controller
 └── Autoplay Controller

EmbeddedVideoPlayer
 ├── Strict State Reset
 ├── No Carryover State
 └── Clean Media Initialization
```

### 🔥 Why this matters:
- Eliminates playback bugs
- Fixes autoplay failures
- Prevents VLC/media desync
- Enables multi-episode stability

## 🎥 Playback Pipeline

```bash
[User Click]
     ↓
[Preload Episode]
     ↓
[Front Probe Validation]
     ↓
[Full Teardown Old Session]
     ↓
[Create New Playback Session]
     ↓
[Start Stream]
     ↓
[Autoplay → Next Episode]
```

💡 Philosophy:
- No reused media state  
- No fake-ready playback  
- Every episode = fresh system

## 🧩 Architecture (V2)

Project Nava uses a **clean playback architecture**:

- 🔹 Full teardown between episodes
- 🔹 No reused media state
- 🔹 Dedicated playback session per episode
- 🔹 Preload → Validate → Play pipeline

This ensures:
- Stability
- No stuck playback
- Reliable autoplay chains
